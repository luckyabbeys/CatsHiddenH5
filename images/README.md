# 图片目录

请在此目录中添加以下文件：

1. `background.jpg` - 游戏的背景图片，建议使用高清图片，最大宽度不超过1920px

2. `cat1.png` 到 `cat10.png` - 10个透明背景的猫咪PNG图片

注意：
- 所有猫咪图片应该使用透明背景（PNG格式）
- 猫咪图片大小建议在50px左右
- 确保图片质量清晰但文件大小适中，以保证加载速度
